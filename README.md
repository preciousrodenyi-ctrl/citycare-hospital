# City Care Hospital Website

## Project Overview
City Care Hospital is a responsive website that provides information about hospital services, allows users to book appointments, and contact the hospital directly. The project aims to demonstrate HTML, CSS, and JavaScript skills with form validation and data persistence.

## Features
- Home, About, Services, and Contact sections
- Appointment booking form with validation
- Contact form with validation
- Data stored in localStorage for appointments and messages
- Responsive design using CSS Flexbox
- Interactive navigation

## Technologies Used
- HTML5
- CSS3
- JavaScript
- LocalStorage for data persistence
- VS Code for development

## How to Use
1. Clone the repository: `git clone <repository-url>`
2. Open `index.html` in a browser
3. Navigate through pages using the navigation menu
4. Fill out the forms to test functionality

## Known Bugs
- Some mobile browsers may have minor styling issues
- Appointment and contact data stored in localStorage is cleared when the browser cache is cleared
